# Projet-2-Contr-le-d-acc-s
Projet Contrôle d'Accès
Présentation : 
Ce projet a pour objectif de concevoir un système de contrôle d'accès sécurisé pour un bâtiment.
La système permet d'identifier les utilisateurs en utilisant plusieurs méthodes d'authentification et de contrôler l'ouverture d'une porte.
